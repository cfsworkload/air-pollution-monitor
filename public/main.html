FIWARE is an emerging standard for designing IoT applications, actively promoted by the European Commission and being increasingly adopted in Smart Cities projects in Europe as well as other areas. Data collected from an extensive set of experimental IoT facilities deployed across Europe is published via NGSI-based APIs surfaced by the FIWARE Orion Broker service. The data can be then used by the various cloud services for storage, processing, analytics, visualization etc, leveraging either subscription or query APIs.

This reference architecture demonstrates how Bluemix can be used to design a powerful cloud-based FIWARE-compliant IoT architecture, comprising FIWARE Orion Broker deployed in FIWARE Lab combined with several Bluemix services used to ingest, process and store the data, and then to use it for advanced visualization and analytics.
